Parte 01:

2. Agora formate as tabelas e deixe o visual como preferir.

3. Para resolver o problema na coluna “Valor do desconto” você deve analisar o seguinte enunciado:

  • Determinada loja está concedendo descontos de 5% para os clientes que realizarem compras superiores a R$ 200,00 e efetuares o pafamento a vista.

4. Na coluna “Valor a ser pago” você deve utilizar uma simples operação matemática, então descubra qual.
