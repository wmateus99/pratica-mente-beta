Atividade de Windows -

1. Abra o Paint e faça um desenho da bandeira do Brasil..

2. Utilize formas geométricas para facilitar. Veja um exemplo abaixo:

3. Após terminar, salve a ilustração na Área de Trabalho com o seguinte nome “Bandeira do Brasil”, e feche o Paint assim que o desenho estiver salvo.

4. Abra o Bloco de Notas e escreva o seguinte poema:

Na frente do PC eu fico a pensar,
Com o teclado começo a digitar.
Faço um site, um jogo ou botão,
Tudo começa com a imaginação.

O HTML é como a estrutura da casa,
CSS é a pintura que arrasa.
JavaScript é o que faz se mexer,
Tipo abrir janela ou fazer chover.

Se o sistema trava, a gente vai lá,
Procura o erro e tenta arrumar.
É igual consertar um carro no escuro,
Mas quando acerta, parece até ouro puro.

Banco de dados é tipo caderno,
Guarda tudo, do começo ao inverno.
Nomes, senhas, tudo com cuidado,
Pra ninguém perder o que foi guardado.

5. Salve o arquivo de texto na Área de Trabalho com algum nome que achar melhor..

6. Mova seu desenho para a pasta "Imagens" no explorador de arquivo.

7. Mova seu arquivo de texto para a pasta "Documentos" no explorador de arquivos.

8. Por fim, exclua de forma permanente os arquivos que você criou.

9. Chame o seu instrutor para verificar se realmente você excluiu permanentemente os arquivos.

10. Assim o exercício estará concluído.
